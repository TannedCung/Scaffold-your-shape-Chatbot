Create a structured streaming response from agent execution.

Processes real-time agent updates and formats them for easy client consumption. Returns structured JSON chunks that clients can easily parse and display.

Parameters:

  • agent_app – The compiled LangGraph agent application

  • initial_state (Dict[str, Any]) – Initial state for agent execution

  • config (Dict[str, Any]) – Agent configuration

  • chat_id (str) – Optional chat completion ID

  • user_id (str) – User ID for context

  • session_id (str) – Session ID for memory management

  • user_message (str) – Original user message for memory storage

Yields:

Formatted JSON strings for streaming response
